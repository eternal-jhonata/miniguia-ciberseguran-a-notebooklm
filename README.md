# 🛡️ Miniguia de Cibersegurança com NotebookLM

## 📖 Sobre o projeto

Este projeto foi desenvolvido como parte do desafio **"Treinando uma IA de Aprendizagem: Explore o Poder do NotebookLM"** da DIO.

O objetivo foi utilizar o NotebookLM como ferramenta de aprendizagem para pesquisar, organizar informações e criar um miniguia sobre cibersegurança utilizando fontes confiáveis.

---

# 🎯 Objetivos

- Aprender a utilizar o NotebookLM;
- Pesquisar informações em fontes confiáveis;
- Organizar o conhecimento utilizando Inteligência Artificial;
- Criar um miniguia de cibersegurança para iniciantes;
- Desenvolver habilidades de Engenharia de Prompts.

---

# 👥 Público-alvo

Usuários iniciantes que desejam aprender boas práticas de segurança digital.

---

# 📚 Fontes utilizadas

Durante o desenvolvimento do projeto foram utilizadas fontes abertas e confiáveis:

- Cartilha de Segurança para Internet (CERT.br)
- Cartilha de Golpes e Fraudes (CERT.br)
- OWASP Top 10 Web Application Security Risks

# 🔄 Melhorando os prompts

Durante o desenvolvimento foi possível perceber que perguntas muito curtas produziam respostas genéricas.

Exemplo:

Prompt inicial:

A melhoria dos prompts gerou respostas mais completas e organizadas.

# 📖 Miniguia de Cibersegurança

## O que é Cibersegurança?

Cibersegurança é o conjunto de práticas utilizadas para proteger computadores, celulares, contas e informações contra ataques digitais.

---

## Principais ameaças

- Ransomware
- Golpes digitais
- Fraudes
- Boatos (Hoaxes)
- Vulnerabilidades em aplicações web

---

## Como se proteger

- Mantenha seus aplicativos atualizados;
- Faça backups regularmente;
- Desconfie de mensagens suspeitas;
- Verifique informações antes de clicar em links;
- Utilize senhas fortes;
- Proteja suas contas.

---

# 📋 Checklist de Segurança

- Atualizar aplicativos
- Fazer backup dos arquivos
- Remover programas desnecessários
- Verificar links antes de clicar
- Desconfiar de golpes
- Manter antivírus atualizado
- Proteger contas
- Aprender sobre novos golpes
- Utilizar fontes confiáveis
- Revisar regularmente a segurança dos dispositivos

---

# 📘 Glossário

**Backup:** cópia de segurança dos arquivos.

**Cibersegurança:** práticas para proteger dispositivos e informações.

**Golpe:** tentativa de enganar o usuário.

**Ransomware:** malware que bloqueia arquivos e exige pagamento.

**Spam:** mensagens enviadas em massa.

**Vulnerabilidade:** falha que pode ser explorada por criminosos.
